En esta sesion practicamos la siguiente actividad:


    Paso 1: Crear en el escritorio una carpeta llamada viernes
    Paso 2: Crear dentro de la carpeta viernes una subcarpeta llamada assets
    Paso 3: Arrasta a Visual Studio Code la carpeta principal que se llama viernes.
    Paso 4: Crea un nuevo archivo llamado index.html y dentro de él, crea la estructura básica de html.
    Paso 5: Llamemos la imágen desde un servidor externo y que se habra en una pestaña nueva utilizando target="_blank"
    target="_blank" : Esto sirve para colocar en un enlace para indicar que la página de destino hacia la que apunta se abra en otra ventana del navegador.
    Paso 6: Llamemos la imágen desde un servidor externo.

Formato de imagenes

.jpe : es el nombre de un comité de expertos que creó un
estándar de compresión y codificación de archivos e imágenes fijas
gif : es un formato de intercambio de imágenes
png : es un formato gráfico basado en un algoritmo de compresión
sin pérdida para bitmaps no sujeto a patentes.

Imágenes

img : no lleva cierre, esto pasa porque su contenido está en el
atributo src, por lo tanto NO todas las etiquetas HTML necesitan un cierre.
